# Foto Kita Blur

A Python project that detects a peace ✌️ hand gesture using MediaPipe and automatically blurs the camera.

## Features

- Real-time webcam
- Peace sign detection
- Gaussian blur effect
- Hand landmark visualization

## Requirements

- Python 3.12
- OpenCV
- MediaPipe

## Installation

```bash
pip install -r requirements.txt
```
  
## Run
```bash
py FotoKitaBlur.py
```

Remember to download and place hand_landmarker.task file in the same folder as FotoKitaBlur.py before running the script
